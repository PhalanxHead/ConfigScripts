# Location Hotkeys AHK Script
Opens common drive locations in Microsoft Explorer or a browser

## Win + H
Opens an input to add the selected string as a replacement for the string you'll input.

## Win + Shift + H
Opens an input to add the selected string to be replaced by the string you'll input.

## Alt + O
Opens my Work OneDrive window.

## Alt + P
Opens the Synced Sharepoint Documents drive.

## Alt + D
Opens the Documents folder.

## Alt + Shift + G
Opens a new Google tab

## Alt + Shift + T 
Opens eTes in a new tab

## Alt + Shift + M 
Opens myDXC in a new tab
